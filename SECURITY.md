# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in BRIG, please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, please email **nabil@happykhan.com** with:

- A description of the vulnerability
- Steps to reproduce the issue
- Any relevant logs or screenshots

You should receive a response within 7 days. We will work with you to understand and address the issue before any public disclosure.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.0.x   | Yes       |
| < 1.0   | No        |
